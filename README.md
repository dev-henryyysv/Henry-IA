HenricoAI é uma inteligência artificial interativa desenvolvida em Python utilizando Kivy, focada em conversação, aprendizado, ensino educacional e interface futurista animada.

O projeto combina:
- IA conversacional
- memória persistente
- sistema de evolução por XP
- base de conhecimento escolar
- integração com Wikipedia
- interface sci-fi dinâmica
- animações e partículas em tempo real

---

# 🚀 Funcionalidades

## 🧠 Inteligência Conversacional
- Conversas naturais
- Respostas contextuais
- Sistema de memória
- Reconhecimento de emoções
- Interação humanizada

## 📚 Base Educacional
A IA possui uma base escolar integrada com conteúdos de:
- Matemática
- Português
- História
- Geografia
- Biologia
- Física
- Química

Separados por:
- Fundamental I
- Fundamental II
- Ensino Médio
- Superior

Com:
- definições
- exemplos
- exercícios
- respostas

---

## 🌐 Pesquisa Inteligente
Integração com Wikipedia:
- pesquisas automáticas
- respostas rápidas
- expansão de conhecimento

---

## ⚡ Sistema de XP e Evolução
O usuário ganha XP conforme utiliza a IA.

Sistema inclui:
- níveis
- barra animada de progresso
- level up
- evolução visual

---

## 💾 Memória Persistente
A IA consegue lembrar:
- nome do usuário
- gostos pessoais
- preferências

Os dados ficam salvos localmente em JSON.
